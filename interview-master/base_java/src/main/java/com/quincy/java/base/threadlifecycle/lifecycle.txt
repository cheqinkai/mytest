生命周期图说明

1，new thread
2，runnable:线程已经被启动，正在等待被分配给CPU时间片，也就是通过调用线程的strat()方法，启动线程，进入就绪状态
            处于就绪状态的线程具备执行条件，但是没有执行权
3，running:线程获得CPU资源正在执行任务，此时除非线程自动放弃CPU资源，或者有优先级更高的线程进入，线程将一直运行到结束
           线程具备执行条件，具有执行权
4，blocked:由于某种原因线程让出CPU资源，并且暂停自己到执行，例如调用sleep(),wait()方法
           线程具备执行条件，不具备执行权
5，wait   会释放锁
6，sleep  不会释放锁
7，dead:线程不具备执行条件和执行权
